Historical Weather Data Api

This api contains weather data records of 6599 from across europen weather stations.

Given below is URL Format and URL Example for how you can enter url to obtain data for specific station on specific date.

  • URL Format: http://127.0.0.1:5000/api/v1/stationId/date

  • URL Example: http://127.0.0.1:5000/api/v1/10/1998-10-25

  • Please enter URL in format given above to avoide getting any erros.

Given below is URL Format and URL Example for how you can enter url to obtain all datat for specific station.

    Due to format reasons the temperature show in this data is multiplied by 10.

  • URL Format: http://127.0.0.1:5000/api/v1/stationId

  • URL Example: http://127.0.0.1:5000/api/v1/10

  • Please enter URL in format given above to avoide getting any erros.

Given below is URL Format and URL Example for how you can enter url to obtain all datat for specific station for a specific year.

    Due to format reasons the temperature show in this data is multiplied by 10.

  • URL Format: http://127.0.0.1:5000/api/v1/yearly/stationId/year

  • URL Example: http://127.0.0.1:5000/api/v1/yearly/10/1998

  • Please enter URL in format given above to avoide getting any erros.
